# Images Directory

This directory should contain your photography images. Here's the recommended structure:

## Required Images

### Core Images
- `logo.svg` - Your photography logo
- `avatar.jpg` - Profile photo (300x300px recommended)
- `placeholder.jpg` - Default placeholder image

### Page Covers
- `golden-hour-cover.jpg` - Cover image for golden hour blog post
- `desert-editorial-cover.jpg` - Cover image for desert editorial post

## Recommended Structure

```
images/
├── logo.svg
├── avatar.jpg
├── placeholder.jpg
├── portfolio/
│   ├── editorial/
│   ├── lifestyle/
│   ├── portraits/
│   └── travel/
└── posts/
    └── (blog post cover images)
```

## Image Specifications

### Portfolio Images
- Format: JPG
- Size: 1920x1280px (3:2 ratio)
- Quality: 85-90%
- Optimized for web

### Blog Covers
- Format: JPG
- Size: 1600x900px (16:9 ratio)
- Quality: 80-85%

### Profile/Avatar
- Format: JPG
- Size: 300x300px (square)
- Quality: 90%

## Tips

1. Use descriptive filenames (e.g., `portrait-session-golden-gate.jpg`)
2. Keep file sizes under 500KB when possible
3. Use image optimization tools before uploading
4. Maintain consistent aspect ratios within categories
